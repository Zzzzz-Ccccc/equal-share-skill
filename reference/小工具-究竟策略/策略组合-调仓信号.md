# 策略组合-调仓信号

> 分类: 究竟策略 | 目录: `小工具-究竟策略` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.JiujingStrategyTool` |
| 方法 | `changeSignal` |
| 版本 | `1.1.0` |

### 描述

获取策略组合调仓信号

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `strategyId` | `java.lang.String` | 是 | - | - |

### 返回值

策略组合调仓信号数据

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `String` | `tradingDay` | 交易日 | 否 |
| `Integer` | `timeFlag` | 时间标记 0：今天 1：明日 2：最新 | 否 |
| `String` | `nextNearestDay` | 最近交易日 | 否 |
| `List<StockInfoVo>` | `buyList` | 买入股票列表 | 否 |
| `List<StockInfoVo>` | `sellList` | 卖出股票列表 | 否 |

### 调用示例

```python
obj = equal.login()

# 调用接口
result = obj.request(
    uri="/api/tool/outcome/strategy/change/signal",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认: 
        "strategyId": None,  # java.lang.String  # 默认: 
    }
)
```
