# 策略组合-组合分析

> 分类: 究竟策略 | 目录: `小工具-究竟策略` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.JiujingStrategyTool` |
| 方法 | `strategyAnalysis` |
| 版本 | `1.1.0` |

### 描述

获取策略组合分析数据

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `strategyId` | `java.lang.String` | 是 | - | - |
| `periodType` | `java.lang.Integer` | 是 | - | - |

### 返回值

策略组合分析数据列表

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `String` | `strategyName` | 策略名称 | 是 |
| `Integer` | `strategyId` | 策略Id | 是 |
| `BigDecimal` | `volatility` | 波动率 | 否 |
| `BigDecimal` | `sharpeRatio` | 夏普率 | 否 |
| `BigDecimal` | `drawdown` | 最大回撤 | 否 |

### 调用示例

```python
obj = equal.login()

# 调用接口
result = obj.request(
    uri="/api/tool/outcome/strategy/analysis",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认: 
        "strategyId": None,  # java.lang.String  # 默认: 
        "periodType": None,  # java.lang.Integer  # 默认: 
    }
)
```
