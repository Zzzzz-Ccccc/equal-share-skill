# 究竟策略最新交易明细

> 分类: 究竟策略 | 目录: `小工具-究竟策略` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.JiujingStrategyTool` |
| 方法 | `getLatestTrades` |
| 版本 | `1.1.0` |

### 描述

获取究竟策略最新交易明细

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `strategyId` | `java.lang.String` | 是 | - | - |

### 返回值

究竟策略最新交易明细列表

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `String` | `orderBookId` | 订单ID（关联买卖） | 否 |
| `String` | `tradingDay` | 交易日 | 否 |
| `String` | `stockCode` | 股票代码 | 是 |
| `String` | `stockName` | 股票名称 | 否 |
| `String` | `side` | 交易方向 buy：开仓 sell：平仓 | 否 |
| `BigDecimal` | `avgPrice` | 成交均价 | 否 |
| `Long` | `quantity` | 成交数量 | 否 |
| `Integer` | `isBonus` | 是否分红 0-否 1-是 | 否 |
| `BigDecimal` | `summaryReturns` | 累计盈亏（sell 时取值） | 否 |
| `BigDecimal` | `holdCost` | 单笔持仓成本（buy 时取值） | 否 |

### 调用示例

```python
obj = equal.login()

# 调用接口
result = obj.request(
    uri="/api/tool/outcome/strategy/latest/trade",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认: 
        "strategyId": None,  # java.lang.String  # 默认: 
    }
)
```
