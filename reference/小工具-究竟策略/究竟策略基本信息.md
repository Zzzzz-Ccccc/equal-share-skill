# 究竟策略基本信息

> 分类: 究竟策略 | 目录: `小工具-究竟策略` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.JiujingStrategyTool` |
| 方法 | `baseInfo` |
| 版本 | `1.1.0` |

### 描述

获取指定策略的基本信息

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `strategyId` | `java.lang.String` | 是 | - | - |

### 返回值

究竟策略基本信息

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `String` | `strategyName` | 策略名称 | 是 |
| `String` | `strategyId` | 策略ID | 是 |
| `String` | `tradingDay` | 交易日 | 否 |
| `BigDecimal` | `recent1Month` | 近1月收益 | 否 |
| `BigDecimal` | `recent3Month` | 近3月收益 | 否 |
| `BigDecimal` | `recent6Month` | 近6月收益 | 否 |
| `BigDecimal` | `recent1Year` | 近1年收益 | 否 |
| `BigDecimal` | `recent3Year` | 近3年收益 | 否 |
| `BigDecimal` | `recent5Year` | 近5年收益 | 否 |
| `BigDecimal` | `yearToDay` | 今年以来收益 | 否 |
| `BigDecimal` | `totalRate` | 累计收益率 | 否 |
| `Integer` | `holdingCount` | 持仓个数 0-空仓 | 否 |
| `Integer` | `createDateNum` | 创建天数 | 否 |
| `String` | `createDate` | 策略创建日期 | 否 |
| `BigDecimal` | `unitNetValue` | 净值 | 否 |
| `BigDecimal` | `dailyReturns` | 日收益率（组合） | 否 |
| `Integer` | `buyNum` | 买入股票数量 | 否 |
| `Integer` | `sellNum` | 卖出股票数量 | 否 |
| `String` | `signalDate` | 调仓信号最新日期 | 否 |
| `BigDecimal` | `bestRatio` | 最好看的收益指标 | 否 |
| `Integer` | `bestRatioType` | 最好看的收益指标类型 | 否 |
| `Integer` | `selectedShowStatus` | 近3年、近5年展示状态 | 否 |
| `String` | `rule` | 调仓规则 | 否 |
| `String` | `investRule` | 投资规则 | 否 |

### 调用示例

```python
obj = equal.login()

# 调用接口
result = obj.request(
    uri="/api/tool/outcome/strategy/base",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认: 
        "strategyId": None,  # java.lang.String  # 默认: 
    }
)
```
