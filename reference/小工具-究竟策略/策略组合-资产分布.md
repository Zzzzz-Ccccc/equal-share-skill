# 策略组合-资产分布

> 分类: 究竟策略 | 目录: `小工具-究竟策略` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.JiujingStrategyTool` |
| 方法 | `strategyAssetAllocationVo` |
| 版本 | `1.1.0` |

### 描述

获取策略组合资产分布数据

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `strategyId` | `java.lang.String` | 是 | - | - |
| `dateStr` | `java.lang.String` | 是 | - | - |

### 返回值

策略组合资产分布数据

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `BigDecimal` | `marketValue` | 持仓市值 | 否 |
| `BigDecimal` | `marketValueRatio` | 持仓市值占比 | 否 |
| `BigDecimal` | `cash` | 现金 | 否 |
| `BigDecimal` | `cashRatio` | 现金占比 | 否 |
| `BigDecimal` | `totalValue` | 总资产 | 否 |
| `BigDecimal` | `totalValueRatio` | 总资产占比 | 否 |
| `BigDecimal` | `todayAddMoney` | 今日转入资金 | 否 |

### 调用示例

```python
obj = equal.login()

# 调用接口
result = obj.request(
    uri="/api/tool/outcome/strategy/asset/allocation",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认: 
        "strategyId": None,  # java.lang.String  # 默认: 
        "dateStr": None,  # java.lang.String  # 默认: 
    }
)
```
