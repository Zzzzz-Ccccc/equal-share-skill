# 策略组合当前持仓

> 分类: 究竟策略 | 目录: `小工具-究竟策略` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.JiujingStrategyTool` |
| 方法 | `currentHolding` |
| 版本 | `1.1.0` |

### 描述

获取策略组合当前持仓列表

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `strategyId` | `java.lang.String` | 是 | - | - |
| `tradeDate` | `java.lang.String` | 是 | - | - |
| `pageIndex` | `java.lang.Integer` | 是 | - | - |
| `pageSize` | `java.lang.Integer` | 是 | - | - |

### 返回值

策略组合当前持仓列表

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `String` | `stockCode` | 股票代码 | 是 |
| `String` | `stockName` | 股票名称 | 否 |
| `BigDecimal` | `holdingRatio` | 持仓比例 | 否 |
| `BigDecimal` | `marketValue` | 持仓最新市值 | 否 |
| `BigDecimal` | `holdingCost` | 持仓总成本 | 否 |
| `BigDecimal` | `holdingCostNumber` | 持仓总成本（未转换） | 否 |
| `BigDecimal` | `dailyPnl` | 当日盈亏 | 否 |
| `Long` | `quantity` | 持仓数量 | 否 |
| `Long` | `quantityNumber` | 持仓数量（未转换） | 否 |
| `Integer` | `holdingDays` | 持仓天数 | 否 |
| `BigDecimal` | `holdingAvgPrice` | 持仓成本均价 | 否 |

### 调用示例

```python
obj = equal.login()

# 调用接口
result = obj.request(
    uri="/api/tool/outcome/strategy/current/holding",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认:

        "tradeDate": None,  # java.lang.String  # 默认: 
        "pageIndex": None,  # java.lang.Integer  # 默认: 
        "pageSize": None,  # java.lang.Integer  # 默认: 
    }
)
```
