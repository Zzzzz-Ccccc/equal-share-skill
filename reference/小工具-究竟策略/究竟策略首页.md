# 究竟策略首页

> 分类: 究竟策略 | 目录: `小工具-究竟策略` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.JiujingStrategyTool` |
| 方法 | `homePage` |
| 版本 | `1.1.0` |

### 描述

获取究竟策略首页数据

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |

### 返回值

究竟策略首页数据

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `String` | `strategyName` | 策略名称 | 是 |
| `String` | `strategyId` | 策略ID | 是 |
| `BigDecimal` | `bestRatio` | 最好看的收益指标 | 否 |
| `BigDecimal` | `netVal` | 最新净值 | 否 |
| `RightsConstant.ModuleRightsBaseInfo` | `moduleRightsBaseInfo` | 权益内容 | 否 |
| `Long` | `buyNum` | 购买数量 | 否 |
| `Long` | `sellNum` | 卖出数量 | 否 |
| `String` | `tradingDate` | 交易日期 | 否 |
| `String` | `nextNearestDay` | 最近交易日 | 否 |
| `BigDecimal` | `dailyReturns` | 当日涨跌 | 否 |
| `Integer` | `timeFlag` | 时间标记 0:今日 1:明日 2:最新 | 否 |
| `String` | `indexCode` | 指标编码 | 否 |
| `String` | `indexName` | 指标名称 | 否 |
| `String` | `redisKey` | 缓存key | 否 |
| `String` | `tradeDetialUrl` | 交易明细跳转链接 | 否 |
| `String` | `strategyDetialUrl` | 策略详情跳转链接 | 否 |
| `String` | `trendChartUrl` | 走势图地址 | 否 |
| `List<String>` | `trendDataList` | 趋势数据列表 | 否 |
| `String` | `tradeRecord` | 交易记录原始串 | 否 |
| `List<StrategyHoldingDetialVo>` | `dealRecordList` | 交易明细列表 | 否 |

### 调用示例

```python
obj = equal.login()

# 调用接口
result = obj.request(
    uri="/api/tool/outcome/strategy/home/page",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认: 
    }
)
```
