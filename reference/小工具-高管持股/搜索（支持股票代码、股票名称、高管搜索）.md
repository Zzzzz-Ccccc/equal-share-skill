# 搜索（支持股票代码、股票名称、高管搜索）

> 分类: 高管持股 | 目录: `小工具-高管持股` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.ManagerHoldingTool` |
| 方法 | `search` |
| 版本 | `1.1.0` |

### 描述

高管持股搜索（支持股票代码、股票名称、高管搜索）

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `keyword` | `java.lang.String` | 是 | - | - |

### 返回值

高管持股搜索结果汇总

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `List<SearchTotalVo>` | `stockList` | 股票搜索结果列表 | 否 |
| `List<SearchTotalVo>` | `peopleList` | 人物搜索结果列表 | 否 |
| `List<SearchTotalVo>` | `agencyList` | 机构搜索结果列表 | 否 |
| `List<SearchTotalVo>` | `fundList` | 基金搜索结果列表 | 否 |

### 调用示例

```python
obj = equal.login()

# 调用接口
result = obj.request(
    uri="/api/tool/manager/holding/search",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认: 
        "keyword": None,  # java.lang.String  # 默认: 
    }
)
```
