# 个股增减持计划详情

> 分类: 高管持股 | 目录: `小工具-高管持股` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.ManagerHoldingTool` |
| 方法 | `getPlanDetailData` |
| 版本 | `1.1.0` |

### 描述

获取个股增减持计划详情数据

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `pageIndex` | `java.lang.Integer` | 是 | - | - |
| `pageSize` | `java.lang.Integer` | 是 | - | - |
| `stockCode` | `java.lang.String` | 是 | - | - |
| `managerName` | `java.lang.String` | 是 | - | - |

### 返回值

个股增减持计划详情列表

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `String` | `stockCode` | 股票代码 | 是 |
| `String` | `stockName` | 股票名称 | 否 |
| `String` | `companyName` | 公司全称 | 否 |
| `String` | `type` | 类型：增持/减持 | 否 |
| `String` | `announcementDate` | 公告日期 | 否 |
| `String` | `pdfUrl` | 公告链接 | 否 |
| `String` | `progress` | 进度(进行中、已完成) | 否 |
| `String` | `holderPositionDesc` | 股东名称以及职位 | 否 |
| `BigDecimal` | `transactionRatio` | 增减持数量占总股本比例 | 否 |
| `List<ManagerSharesPlanDetailVo.ManagerInfo>` | `managerInfoList` | 计划涉及的高管信息列表 | 否 |
| `Integer` | `managerInfoList.personEsId` | 高管ESId | 否 |
| `String` | `managerInfoList.personInfoId` | 高管InfoId | 否 |
| `String` | `managerInfoList.managerName` | 高管姓名 | 否 |
| `String` | `managerInfoList.position` | 职务 | 否 |

### 调用示例

```python
obj = equal.login()

# 调用接口
result = obj.request(
    uri="/api/tool/manager/holding/{stockCode}/plan/detail",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认: 
        "pageIndex": None,  # java.lang.Integer  # 默认: 
        "pageSize": None,  # java.lang.Integer  # 默认: 
        "stockCode": None,  # java.lang.String  # 默认: 
        "managerName": None,  # java.lang.String  # 默认: 
    }
)
```
