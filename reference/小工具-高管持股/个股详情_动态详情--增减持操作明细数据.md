# 个股详情/动态详情--增减持操作明细数据

> 分类: 高管持股 | 目录: `小工具-高管持股` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.ManagerHoldingTool` |
| 方法 | `getChangeDetailData` |
| 版本 | `1.1.0` |

### 描述

获取个股增减持操作明细数据

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `pageIndex` | `java.lang.Integer` | 是 | - | - |
| `pageSize` | `java.lang.Integer` | 是 | - | - |
| `changeType` | `java.lang.Integer` | 是 | - | - |
| `stockCode` | `java.lang.String` | 是 | - | - |
| `managerName` | `java.lang.String` | 是 | - | - |
| `changeDate` | `java.lang.String` | 是 | - | - |

### 返回值

个股增减持操作明细列表

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `String` | `stockCode` | 股票代码 | 是 |
| `String` | `stockName` | 股票名称 | 否 |
| `Long` | `totalQuantity` | 增减持数量 | 否 |
| `BigDecimal` | `totalAmount` | 总金额 | 否 |
| `BigDecimal` | `ratioOfTotal` | 占总股本比 | 否 |
| `BigDecimal` | `ratioOfCirculate` | 占流通股比 | 否 |
| `Long` | `actualChangeQuantity` | 净增持/减持数量 | 否 |
| `BigDecimal` | `avgPrice` | 增减持均价 | 否 |
| `BigDecimal` | `chgPct` | 当日涨跌幅 | 否 |
| `String` | `changeType` | 持股变动类型 增持/减持 | 否 |
| `String` | `mangerName` | 高管姓名 | 否 |
| `String` | `changeName` | 变动人姓名 | 否 |
| `String` | `position` | 高管职位 | 否 |
| `String` | `relationDesc` | 变动人与高管关系 | 否 |
| `BigDecimal` | `zdOdds` | 上涨概率 | 否 |
| `Integer` | `trackDayNum` | 跟踪天数 | 否 |
| `String` | `changeDate` | 变动日期 | 否 |
| `String` | `changeReason` | 变动原因（增减持方式） | 否 |
| `Integer` | `kinshipNum` | 是否有人物关系图 | 否 |

```python
obj = equal.login()

    uri="/api/tool/manager/holding/{stockCode}/opt/detail",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认:

        "pageSize": None,  # java.lang.Integer  # 默认: 
        "changeType": None,  # java.lang.Integer  # 默认: 
        "stockCode": None,  # java.lang.String  # 默认: 
        "managerName": None,  # java.lang.String  # 默认: 
        "changeDate": None,  # java.lang.String  # 默认: 
    }
)
```
