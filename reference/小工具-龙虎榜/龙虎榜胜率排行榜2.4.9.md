# 龙虎榜胜率排行榜2.4.9

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.DragonTool` |
| 方法 | `getWinRateRank` |
| 版本 | `1.1.0` |

### 描述

获取龙虎榜游资胜率排行榜

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `sortFlag` | `java.lang.Integer` | 是 | - | - |
| `listNum` | `java.lang.Integer` | 是 | - | - |
| `isNew` | `java.lang.Integer` | 是 | - | - |

### 返回值

龙虎榜游资胜率排行榜列表

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `String` | `infoId` | 游资infoId | 否 |
| `Integer` | `infoType` | 信息类型 | 否 |
| `String` | `hotMoney` | 游资名称 | 是 |
| `Integer` | `sort` | 排序序号 | 否 |
| `BigDecimal` | `riseOdds` | 上涨概率 | 否 |
| `BigDecimal` | `avgOdds` | 平均涨幅 | 否 |
| `Integer` | `trackDayNum` | 跟踪天数 | 否 |
| `Integer` | `traceMonth` | 跟踪月数 | 否 |
| `Integer` | `listNum` | 上榜次数 | 否 |
| `String` | `tradeDay` | 交易日期 | 否 |
| `String` | `tradeDayDesc` | 交易日描述（今日/昨日/具体日期） | 否 |
| `List<HotMoneyListSearchVo>` | `tradeRecordList` | 操作买入的股票信息列表 | 否 |
