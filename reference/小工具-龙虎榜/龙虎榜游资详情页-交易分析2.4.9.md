# 龙虎榜游资详情页-交易分析2.4.9

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.DragonTool` |
| 方法 | `getTradeAnalysis` |
| 版本 | `1.1.0` |

### 描述

获取指定游资的交易分析数据

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `hotMoney` | `java.lang.String` | 是 | - | - |

### 返回值

游资交易分析数据，包括最佳上涨概率、平均涨幅及图表数据

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `BigDecimal` | `odds` | 最佳上涨概率 | 否 |
| `BigDecimal` | `zdfAvg` | 最佳平均涨幅 | 否 |
| `Integer` | `riseTrackDay` | 上涨概率跟踪天数 | 否 |
| `Integer` | `zdfAvgTrackDay` | 涨跌幅跟踪天数 | 否 |
| `Integer` | `listNum` | 上买榜次数 | 否 |
| `List<DragonTradeAnalysisVO.AnalysisChartData>` | `chartDataList` | 图表数据列表 | 否 |
| `String` | `chartDataList.tradeDateDesc` | 交易日描述（如T、T+1等） | 否 |
| `BigDecimal` | `chartDataList.riseRatio` | 上涨概率 | 否 |
| `Integer` | `chartDataList.riseNum` | 上升个股数 | 否 |
| `Integer` | `chartDataList.fallNum` | 下跌个股数 | 否 |
| `BigDecimal` | `chartDataList.zdfAvg` | 平均涨幅 | 否 |

### 调用示例

```python
obj = equal.login()

# 调用接口
result = obj.request(
    uri="/api/tool/dragon/tiger/detail/{hotMoney}/analysis",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认: 
        "hotMoney": None,  # java.lang.String  # 默认: 
    }
)
```
