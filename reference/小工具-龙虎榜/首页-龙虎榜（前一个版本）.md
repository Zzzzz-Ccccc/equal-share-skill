# 首页-龙虎榜（前一个版本）

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.DragonTool` |
| 方法 | `homepagePrevious` |
| 版本 | `1.1.0` |

### 描述

首页-龙虎榜（前一个版本），返回龙虎榜首页数据（兼容旧版）

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |

### 返回值

龙虎榜首页数据（旧版本兼容接口），包含首页展示所需的汇总信息与列表信息

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `object` | `homepageSummary` | 首页概览统计信息（如成交额、上榜数量等） | 否 |
| `array` | `homepageSections` | 首页各模块数据列表，如热门个股榜、热门游资榜等 | 否 |

### 调用示例

```python
obj = equal.login()

# 调用接口
result = obj.request(
    uri="/api/tool/dragon/tiger/homepage",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认: 
    }
)
```
