# 龙虎榜日期列表

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.DragonTool` |
| 方法 | `dateList` |
| 版本 | `1.1.0` |

### 描述

获取龙虎榜可用的历史交易日期列表

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |

### 返回值

龙虎榜日期列表，按时间倒序或业务规则排序

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `List<String>` | `tradeDates` | 可查询的龙虎榜交易日期列表 | 是 |

```python
obj = equal.login()

# 调用接口
```
