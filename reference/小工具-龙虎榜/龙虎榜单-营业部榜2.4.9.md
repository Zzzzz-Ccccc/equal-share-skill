# 龙虎榜单-营业部榜2.4.9

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.DragonTool` |
| 方法 | `getDeptRank` |
| 版本 | `1.1.0` |

### 描述

获取指定交易日的龙虎榜营业部榜单

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `tradeDate` | `java.lang.String` | 是 | - | - |

### 返回值

指定交易日的营业部榜数据列表

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `String` | `deptName` | 营业部名称 | 是 |
| `List<String>` | `hotMoneyNames` | 关联游资名称列表 | 否 |
| `List<DeptListVo.StockInfo>` | `buyStockList` | 买入股票列表 | 否 |
| `List<DeptListVo.StockInfo>` | `sellStockList` | 卖出股票列表 | 否 |
| `Integer` | `buyNum` | 买入个股数量 | 否 |
| `Integer` | `sellNum` | 卖出个股数量 | 否 |
| `BigDecimal` | `buyAmount` | 买入金额 | 否 |
| `BigDecimal` | `sellAmount` | 卖出金额 | 否 |
| `BigDecimal` | `buyAmountClean` | 净买入金额 | 否 |
| `String` | `buyStockList.stockCode` | 买入股票代码 | 否 |
| `String` | `buyStockList.stockName` | 买入股票名称 | 否 |
| `Integer` | `buyStockList.periodDays` | 买入连续N日标签 | 否 |
| `String` | `sellStockList.stockCode` | 卖出股票代码 | 否 |
| `String` | `sellStockList.stockName` | 卖出股票名称 | 否 |
| `Integer` | `sellStockList.periodDays` | 卖出连续N日标签 | 否 |

### 调用示例

```python
obj = equal.login()

# 调用接口
result = obj.request(
    uri="/api/tool/dragon/tiger/dept/{tradeDate}/rank",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认: 
        "tradeDate": None,  # java.lang.String  # 默认: 
    }
)
```
