# 游资近期操作2.4.9

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.DragonTool` |
| 方法 | `getHotMoneyOpeDetail` |
| 版本 | `1.1.0` |

### 描述

查询游资在指定交易日附近的近期操作明细

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `hotMoneyName` | `java.lang.String` | 是 | - | - |
| `tradeDate` | `java.lang.String` | 是 | - | - |

### 返回值

游资近期操作汇总以及个股明细

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `String` | `hotMoneyName` | 游资名称 | 是 |
| `Integer` | `listNum` | 上榜次数 | 否 |
| `Integer` | `buyNum` | 买入个股数量 | 否 |
| `Integer` | `sellNum` | 卖出个股数量 | 否 |
| `BigDecimal` | `buyAmountClean` | 净买入金额 | 否 |
| `List<HotMoneyOperateVo.DataVo>` | `children` | 近期操作个股明细列表 | 否 |
| `String` | `children.stockCode` | 股票代码 | 否 |
| `String` | `children.stockName` | 股票名称 | 否 |
| `BigDecimal` | `children.buyAmount` | 买入金额 | 否 |
| `BigDecimal` | `children.sellAmount` | 卖出金额 | 否 |
| `BigDecimal` | `children.buyAmountClean` | 净买入金额 | 否 |
| `BigDecimal` | `children.pctChg` | 当日涨跌幅 | 否 |
| `Integer` | `children.periodDays` | 连续N个交易日 | 否 |

obj = equal.login()

    }
