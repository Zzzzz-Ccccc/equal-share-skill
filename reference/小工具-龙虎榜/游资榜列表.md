# 游资榜列表

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.DragonTool` |
| 方法 | `hotMoneyList` |
| 版本 | `1.1.0` |

### 描述

根据交易日期获取游资榜列表

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `tradeDate` | `java.lang.String` | 是 | - | - |

### 返回值

指定交易日的游资榜列表

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `String` | `hotMoneyName` | 游资名称 | 是 |
| `BigDecimal` | `buyAmountClean` | 游资净买入额（原值） | 否 |
| `List<HotMoneyListVo.DataVo>` | `children` | 该游资当日操作的个股明细列表 | 否 |
| `String` | `jumpUrl` | 跳转URL | 否 |
| `String` | `children.stockCode` | 个股代码 | 否 |
| `String` | `children.stockName` | 个股名称 | 否 |
| `BigDecimal` | `children.pctChg` | 个股涨跌幅 | 否 |
| `BigDecimal` | `children.buyAmount` | 买入金额 | 否 |
| `BigDecimal` | `children.sellAmount` | 卖出金额 | 否 |
| `BigDecimal` | `children.buyAmountClean` | 净买入金额 | 否 |
| `BigDecimal` | `children.buyAmountCleanNumber` | 净买入金额原始值 | 否 |
| `Integer` | `children.periodDays` | 连续N个交易日 | 否 |
| `List<String>` | `children.hotMoneyList` | 该股票其他游资操作列表 | 否 |

### 调用示例

```python
obj = equal.login()

# 调用接口
result = obj.request(
    uri="/api/tool/dragon/tiger/hot/money/list",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认: 
        "tradeDate": None,  # java.lang.String  # 默认: 
    }
)
```
