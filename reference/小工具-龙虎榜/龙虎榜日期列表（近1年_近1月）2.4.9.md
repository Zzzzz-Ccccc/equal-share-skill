# 龙虎榜日期列表（近1年/近1月）2.4.9

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.DragonTool` |
| 方法 | `getRecentDateList` |
| 版本 | `1.1.0` |

### 描述

获取龙虎榜近1年或近1月的交易日期列表

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `flag` | `java.lang.Integer` | 是 | - | - |

### 返回值

指定范围内的龙虎榜交易日期列表

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `List<String>` | `tradeDates` | 交易日期列表 | 是 |

### 调用示例

```python
obj = equal.login()

# 调用接口
result = obj.request(
    uri="/api/tool/dragon/tiger/recent/trade_date/list",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认: 
        "flag": None,  # java.lang.Integer  # 默认: 
    }
)
```
