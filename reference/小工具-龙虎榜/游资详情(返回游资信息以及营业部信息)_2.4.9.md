# 游资详情(返回游资信息以及营业部信息) 2.4.9

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.DragonTool` |
| 方法 | `getHotMoneyDetail` |
| 版本 | `1.1.0` |

### 描述

查询游资详情，返回游资信息及相关营业部信息

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `hotMoneyName` | `java.lang.String` | 是 | - | - |

### 返回值

游资详情数据，包含游资统计与相关营业部信息

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `String` | `hotMoneyName` | 游资名称 | 是 |
| `BigDecimal` | `buyAmountClean` | 净买入金额 | 否 |
| `List<HotMoneyListVo.DataVo>` | `children` | 游资操作个股明细列表 | 否 |

### 调用示例

```python
obj = equal.login()

# 调用接口
result = obj.request(
    uri="/api/tool/dragon/tiger/hot/money/dtl249",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认: 
        "hotMoneyName": None,  # java.lang.String  # 默认: 
    }
)
```
