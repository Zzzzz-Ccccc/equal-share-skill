# 营业部所属游资席位

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.DragonTool` |
| 方法 | `deptBelongHotMoney` |
| 版本 | `1.1.0` |

### 描述

根据营业部全称查询所属游资席位列表

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `deptName` | `java.lang.String` | 是 | - | - |

### 返回值

营业部对应的游资席位名称列表

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `List<String>` | `hotMoneyNames` | 游资席位名称列表 | 是 |

### 调用示例

```python
obj = equal.login()

# 调用接口
result = obj.request(
    uri="/api/tool/dragon/tiger/dept/belong/hot/money",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认: 
        "deptName": None,  # java.lang.String  # 默认: 
    }
)
```
