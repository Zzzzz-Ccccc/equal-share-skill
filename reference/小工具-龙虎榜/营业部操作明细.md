# 营业部操作明细

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.DragonTool` |
| 方法 | `deptHoldingDetialList` |
| 版本 | `1.1.0` |

### 描述

查询指定营业部的历史操作明细

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `deptName` | `java.lang.String` | 是 | - | - |
| `sortName` | `java.lang.Integer` | 是 | - | - |
| `sortType` | `java.lang.Integer` | 是 | - | - |
| `pageIndex` | `java.lang.Integer` | 是 | - | - |
| `pageSize` | `java.lang.Integer` | 是 | - | - |

### 返回值

营业部操作明细列表

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `String` | `stockCode` | 股票代码 | 是 |
| `String` | `stockName` | 股票名称 | 否 |
| `String` | `tradeDate` | 交易日期 | 否 |
| `BigDecimal` | `buyAmount` | 买入金额 | 否 |
| `BigDecimal` | `sellAmount` | 卖出金额 | 否 |
| `BigDecimal` | `buyAmountClean` | 净买入金额 | 否 |
| `Integer` | `periodDays` | 连续N个交易日 | 否 |
