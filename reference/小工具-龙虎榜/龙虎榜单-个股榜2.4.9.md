# 龙虎榜单-个股榜2.4.9

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.DragonTool` |
| 方法 | `getStockRank` |
| 版本 | `1.1.0` |

### 描述

获取指定交易日的龙虎榜个股榜单

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `tradeDate` | `java.lang.String` | 是 | - | - |

### 返回值

指定交易日的个股龙虎榜数据列表

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `String` | `jumpUrl` | 跳转链接 | 否 |
| `String` | `stockCode` | 股票代码 | 是 |
| `String` | `stockName` | 股票名称 | 否 |
| `BigDecimal` | `pctChg` | 今日涨幅 | 否 |
| `BigDecimal` | `buyMoneyClean` | 净买入额 | 否 |
| `String` | `tradeDay` | 交易日 | 否 |
| `String` | `hotMoneyNameDesc` | 知名游资(空格隔开) | 否 |
| `Integer` | `periodDays` | 连续N日的标签 | 否 |
| `Integer` | `totalCount` | 统计的总交易日数 | 否 |
| `Integer` | `limitUpCount` | 涨停交易日数量 | 否 |
| `List<String>` | `hotMoneyList` | 游资相关信息列表 | 否 |

### 调用示例

```python
obj = equal.login()

# 调用接口
result = obj.request(
    uri="/api/tool/dragon/tiger/stock/{tradeDate}/rank",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认: 
        "tradeDate": None,  # java.lang.String  # 默认: 
    }
)
```
