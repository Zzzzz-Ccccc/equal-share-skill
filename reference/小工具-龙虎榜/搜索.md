# 搜索

> 分类: 龙虎榜 | 目录: `小工具-龙虎榜` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.DragonTool` |
| 方法 | `search` |
| 版本 | `1.1.0` |

### 描述

根据名称搜索游资或营业部

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `name` | `java.lang.String` | 是 | - | - |

### 返回值

搜索结果列表，包括游资/营业部及相关股票信息

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `String` | `name` | 名称（游资或营业部） | 是 |
| `String` | `shortName` | 简称 | 否 |
| `Integer` | `type` | 类型 1-游资 2-营业部 | 否 |
| `String` | `stockCode` | 股票代码 | 否 |
| `String` | `stockCodeShow` | 展示用股票代码 | 否 |
| `String` | `infoId` | infoId | 否 |
| `Integer` | `dataType` | 数据类型 1-游资或营业部 2-上市公司 | 否 |

### 调用示例

```python
obj = equal.login()

# 调用接口
result = obj.request(
    uri="/api/tool/dragon/tiger/search",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认: 
        "name": None,  # java.lang.String  # 默认: 
    }
)
```
