# 首页机构动向

> 分类: 机构动向 | 目录: `小工具-机构动向` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.AgencyTrendTool` |
| 方法 | `homePage` |
| 版本 | `1.1.0` |

### 描述

首页机构动向

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |

### 返回值

首页机构动向 返回数据

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `String` | `showAgencyName` | 显示的机构名称 | 否 |
| `String` | `agencyName` | 机构名称 | 否 |
| `String` | `fundCompany` | 基金公司 | 否 |
| `String` | `companyShortName` | 公司简称 | 否 |
| `String` | `stockCode` | 股票代码 | 是 |
| `String` | `stockName` | 股票名称 | 否 |
| `String` | `publicDate` | 更新日期 | 否 |
| `Long` | `holderNum` | 持股数量 | 否 |
| `String` | `holderCode` | 股东编码 | 否 |
| `String` | `holderName` | 股东名称 | 否 |
| `String` | `jumpUrl` | 跳转链接 | 否 |
| `Long` | `changeNum` | 变动股数 | 否 |
| `Integer` | `changeDirect` | 持股变动方向 1-新进 2-退出 3-增持 4-减持 | 否 |
| `BigDecimal` | `ratio` | 持股比例 | 否 |
| `Integer` | `agencyType` | 机构类型 | 否 |
| `String` | `agencyOnlyId` | 机构唯一id | 否 |
| `Integer` | `type` | 比例类型 0-总股本 1-流通股 | 否 |
| `JSONArray` | `typesArr` | 类型数组 | 否 |

### 调用示例

```python
obj = equal.login()

# 调用接口
result = obj.request(
    uri="/api/tool/agency/home/page",
    params={
        "request": None,  # jakarta.servlet.http.HttpServletRequest  # 默认: 
    }
)
```
