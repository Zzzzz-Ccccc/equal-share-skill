# 机构本期持股列表

> 分类: 机构动向 | 目录: `小工具-机构动向` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.AgencyTrendTool` |
| 方法 | `currentList` |
| 版本 | `1.1.0` |

### 描述

机构本期持股列表

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `agencyOnlyId` | `java.lang.String` | 是 | - | - |
| `holderName` | `java.lang.String` | 是 | - | - |
| `branchInfoId` | `java.lang.String` | 是 | - | - |
| `pageIndex` | `java.lang.Integer` | 是 | - | - |
| `pageSize` | `java.lang.Integer` | 是 | - | - |
| `sort` | `java.lang.Integer` | 是 | - | - |

### 返回值

机构本期持股列表 返回数据

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `String` | `stockCode` | 股票代码 | 是 |
| `String` | `stockName` | 股票名称 | 否 |
| `BigDecimal` | `marketValue` | 现市值 | 否 |
| `Long` | `holderNum` | 现持股数 | 否 |
| `Long` | `holderNumOld` | 原持股数 | 否 |
| `Long` | `changeNum` | 主动变动股数 | 否 |
| `String` | `changeNumStr` | 变动股数（单位自适应） | 否 |
| `String` | `holderNumStr` | 现持股数（单位自适应） | 否 |
| `BigDecimal` | `holdingCost` | 持股成本 | 否 |
| `BigDecimal` | `endTermPrice` | 期末价格 | 否 |
| `BigDecimal` | `priceAvg` | 均价 | 否 |
| `String` | `reportDate` | 变动期（自匹配季度） | 否 |
| `BigDecimal` | `rateRatio` | 预估收益率 | 否 |
| `Integer` | `isBeforeList` | 是否上市前就持有公司股票 0-否 1-是 | 否 |
| `BigDecimal` | `holdingAvgPrice` | 持仓成本均价 | 否 |
| `String` | `publicDate` | 公告日期 | 否 |
| `Integer` | `changeDirect` | 变动方向 | 否 |
| `String` | `dataDate` | 数据日期 | 否 |
| `String` | `holderName` | 股东名称 | 否 |
| `String` | `agencyOnlyId` | 机构唯一ID | 否 |
| `String` | `holderOnlyStr` | 股东唯一标识串 | 否 |
| `Integer` | `agencyType` | 机构类型 | 否 |
| `Integer` | `type` | 比例类型 0-总股本 1-流通股 | 否 |
| `BigDecimal` | `ratio` | 持股比例 | 否 |
| `String` | `holderCode` | 股东编码 | 否 |
| `String` | `showHolderName` | 展示用股东名称 | 否 |
| `String` | `holderInfoId` | 股东infoId | 否 |
| `String` | `fundStockQuarter` | 基金持仓季度 | 否 |
| `String` | `fundPositionRatio` | 基金持仓比例（占净值比例） | 否 |
