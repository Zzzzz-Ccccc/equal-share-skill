# 机构风云榜-榜单首页2.5.0

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.AgencyTrendTool` |
| 方法 | `getRankLaunchPageList` |
| 版本 | `1.1.0` |

### 描述

机构风云榜-榜单首页2.5.0

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `agencyType` | `java.lang.Integer` | 是 | - | - |
| `pageIndex` | `java.lang.Integer` | 是 | - | - |
| `pageSize` | `java.lang.Integer` | 是 | - | - |
| `firstCategory` | `java.lang.Integer` | 是 | - | - |

### 返回值

机构风云榜-榜单首页2.5.0 返回数据

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `Integer` | `agencyId` | 机构ID | 是 |
| `Integer` | `agencyType` | 榜单机构类型 | 是 |
| `String` | `agencyName` | 机构名称 | 是 |
| `String` | `reportDate` | 报告期 yyyy-MM-dd | 是 |
| `Integer` | `transDirect` | 操作类型  1-买入榜单 2-卖出榜单 | 是 |
| `String` | `reportYear` | 报告期年 | 否 |
| `String` | `reportQuarter` | 报告期季度 | 否 |
| `String` | `staticPicUrl` | 静态图片地址 | 否 |
| `Integer` | `sourceId` | 来源 1-机构动向 2-股东协同 | 否 |
| `Integer` | `firstCategory` | 一级分类 0-热门榜单 1-买入榜 2-卖出榜 3-协同榜 | 否 |
| `String` | `title` | 类型标题 | 否 |
| `String` | `iconUrl` | 榜单类型对应的 icon地址 | 否 |
| `String` | `jumpUrl` | 跳转页面地址（已拼接参数） | 否 |
| `String` | `updateDate` | 数据更新日期 | 否 |
