# 机构历史个股追踪

> 分类: 机构动向 | 目录: `小工具-机构动向` | 索引见 `skill.md`

## 元信息

| 属性        | 值   |
|------------|------|
| 类名 | `cn.lunhui.tiger.ai.skill.basic.AgencyTrendTool` |
| 方法 | `holdingTrack` |
| 版本 | `1.1.0` |

### 描述

机构历史个股追踪

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |
|---------|--------|--------- |---------|------|
| `request` | `jakarta.servlet.http.HttpServletRequest` | 是 | - | - |
| `agencyOnlyId` | `java.lang.String` | 是 | - | - |
| `yearNum` | `java.lang.Integer` | 是 | - | - |
| `holderName` | `java.lang.String` | 是 | - | - |
| `branchInfoId` | `java.lang.String` | 是 | - | - |

### 返回值

机构历史个股追踪 返回数据

| 参数类型   | 字段名称    | 字段说明       | 是否必返回 |
|----------|----------==|-------------- |------------|
| `Integer` | `num` | 跟踪总次数 | 否 |
| `Integer` | `changeDirect` | 持股变动方向 1-新进 2-退出 3-增加 4-减少 5-重仓 | 否 |
| `Integer` | `riseNumTwenty` | 20日上涨次数 | 否 |
| `Integer` | `fallNumTwenty` | 20日下跌次数 | 否 |
| `Integer` | `riseNumForty` | 40日上涨次数 | 否 |
| `Integer` | `fallNumForty` | 40日下跌次数 | 否 |
| `Integer` | `riseNumSixty` | 60日上涨次数 | 否 |
| `Integer` | `fallNumSixty` | 60日下跌次数 | 否 |
| `Integer` | `trackDayNum` | 概率最高的天数 | 否 |
| `String` | `publicDate` | 最近统计日期 | 否 |
| `Boolean` | `isAdvantage` | 是否亮点 | 否 |
