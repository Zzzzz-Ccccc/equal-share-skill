### 描述

### 参数

| 参数名   | 类型    | 必填     | 默认值    | 描述 |

| `agencyOnlyId` | `java.lang.String` | 是 | - | - |

| `String` | `agencyOnlyId` | 机构唯一id | 否 |

| `Date` | `createTime` | 创建时间 | 否 |
| `Date` | `updateTime` | 更新时间 | 否 |
