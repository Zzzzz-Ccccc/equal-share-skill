# 机构持仓列表

> 分类: 究竟特色数据 | 目录: `究竟特色数据` | 索引见 `SKILL.md`


### 描述

机构持仓列表

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `str` | 是 | "DRAGON_TIGER_AGENCY_HOLDER_STOCK_LIST_986A9741" | 接口ID |
| `agencyOnlyId` | `str` | 否 | - | agencyOnlyId |
| `dateStr` | `str` | 是 | - | 日期 |
| `pageIndex` | `int` | 是 | - | 当前页从0开始 |
| `pageSize` | `int` | 是 | - | 每页显示记录数 |
