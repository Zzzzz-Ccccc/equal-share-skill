# 机构详情

> 分类: 究竟特色数据 | 目录: `究竟特色数据` | 索引见 `SKILL.md`


### 描述

机构详情

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `str` | 是 | "DRAGON_TIGER_AGENCY_DETAIL_400A302A" | 接口ID |
| `agencyOnlyId` | `str` | 是 | - | agencyOnlyId |
| `agencyName` | `str` | 是 | - | 机构名称 |
| `agencyType` | `int` | 是 | - | 机构类型 |

### 返回参数

机构动向-机构详情 返回数据

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `agencyName` | `str` | 是 | 机构名称 |
| `fundCompanyName` | `str` | 否 | 基金公司名称 |
| `headImage` | `str` | 否 | 机构头像地址 |
| `synopsis` | `str` | 否 | 机构简介 |
| `agencyTypesArr` | `list` | 否 | 机构类型数组 |
| `fundCompanyTypesArr` | `list` | 否 | 基金公司类型数组 |
| `infoId` | `str` | 否 | 机构infoId |
| `infoType` | `int` | 否 | 机构信息类型 |
| `agencyType` | `int` | 否 | 机构类型代码 |

### 调用示例

```python
from equal_data import KjjApi 
api = KjjApi('bear your token') 
# 调用接口
data = api.query_kjj_data(
    interfaceId="DRAGON_TIGER_AGENCY_DETAIL_400A302A",
    agencyOnlyId=None,  # str  # 默认: 
    agencyName=None,  # str  # 默认: 
    agencyType=None,  # int  # 默认: 
)
```
