# 机构调研列表

### 描述

机构调研列表

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `str` | 是 | "DRAGON_TIGER_AGENCY_RESEARCH_LIST_BD40F7FC" | 接口ID |
| `stockCode` | `str` | 否 | - | 股票代码 |
| `dateStr` | `str` | 否 | - | 公告日期 |
| `pageIndex` | `int` | 是 | - | 当前页 从0开始 |
| `pageSize` | `int` | 是 | - | 每页容量 |

### 返回参数

机构调研列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `str` | 否 | 股票代码 |
| `stockName` | `str` | 否 | 股票名称 |
| `researchDate` | `str` | 否 | 调研日期 |
| `reportDate` | `str` | 否 | 公告日期 |
| `researchFirstTradeDate` | `str` | 否 | 调研后第一个交易日 |
| `starNum` | `int` | 否 | 星标数量 |
