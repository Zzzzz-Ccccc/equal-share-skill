# 机构动向列表数据

> 分类: 究竟特色数据 | 目录: `究竟特色数据` | 索引见 `SKILL.md`


### 描述

机构动向列表数据

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `str` | 是 | "DRAGON_TIGER_AGENCY_PAGE_LIST_V250_97962782" | 接口ID |
| `pageIndex` | `int` | 是 | - | 当前页从0开始 |
| `pageSize` | `int` | 是 | - | 每页显示记录数 |
| `agencyType` | `int` | 否 | - | 机构类型 3001-公募 1003-私募 1006-信托公司 1007-证券公司 <br>1009-保险公司 1015-外资 8000-国家队 1201-知名机构  全部不传值  |
| `changeDirec` | `int` | 否 | - | 持股变动类型 1-新进 2-退出 3-增持 4-减持  全部不传值  |
