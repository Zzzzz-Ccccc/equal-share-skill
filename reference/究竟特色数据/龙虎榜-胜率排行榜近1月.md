# 龙虎榜-胜率排行榜近1月

> 分类: 究竟特色数据 | 目录: `究竟特色数据` | 索引见 `SKILL.md`


### 描述

获取龙虎榜游资胜率排行榜

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `str` | 是 | "DRAGON_TIGER_WIN_RATE_RANK_1AFFA65" | 接口ID |
| `sortFlag` | `int` | 否 | `0` | 排序标记（0 默认） |
| `listNum` | `int` | 否 | `10` | 榜单数量，默认10 |
| `isNew` | `int` | 否 | `0` | 是否仅看新上榜（0 否） |

### 返回参数

龙虎榜-游资胜率排行榜列表

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `infoId` | `str` | 否 | 游资infoId |
| `infoType` | `int` | 否 | 信息类型 |
| `hotMoney` | `str` | 是 | 游资名称 |
| `sort` | `int` | 否 | 排序序号 |
| `riseOdds` | `float` | 否 | 上涨概率 |
| `avgOdds` | `float` | 否 | 平均涨幅 |
| `trackDayNum` | `int` | 否 | 跟踪天数 |
| `traceMonth` | `int` | 否 | 跟踪月数 |
| `listNum` | `int` | 否 | 上榜次数 |
| `tradeDay` | `str` | 否 | 交易日期 |
| `tradeDayDesc` | `str` | 否 | 交易日描述（今日/昨日/具体日期） |
| `tradeRecordList` | `list[HotMoneyListSearchVo]` | 否 | 操作买入的股票信息列表 |

### 调用示例

```python
from equal_data import KjjApi 
api = KjjApi('bear your token') 
# 调用接口
data = api.query_kjj_data(
    interfaceId="DRAGON_TIGER_WIN_RATE_RANK_1AFFA65",
    sortFlag=None,  # int  # 默认: 0
    listNum=None,  # int  # 默认: 10
    isNew=None,  # int  # 默认: 0
)
```
