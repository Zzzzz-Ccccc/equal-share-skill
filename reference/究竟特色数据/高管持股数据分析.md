### 描述

高管持股数据分析

### 请求参数

| 名称 | 类型 | 必填 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| `interfaceId` | `str` | 是 | "DRAGON_TIGER_MANAGER_HOLDING_STOCKCODE_BASIC_INFO_937D50E" | 接口ID |
| `stockCode` | `str` | 是 | - | 股票代码 |

### 返回参数

高管持股数据分析

| 名称 | 类型 | 是否必返回 | 说明 |
|:---|:---|:---|:---|
| `stockCode` | `str` | 是 | 股票代码 |
| `stockName` | `str` | 否 | 股票名称 |
| `companyName` | `str` | 否 | 公司全称 |
| `totalQuantity` | `int` | 否 | 增减持数量 |
| `totalAmount` | `float` | 否 | 总金额 |
| `ratioOfTotal` | `float` | 否 | 占总股本比 |
| `ratioOfCirculate` | `float` | 否 | 占流通股比 |
